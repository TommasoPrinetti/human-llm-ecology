# Sub-Agents

Specialist contracts used by the agent harness.

Each sub-agent folder contains a `SOUL.md` file:

- `conceptualizer/` - search concepts and sequence planning.
- `navigator/` - Realm and Root Vault search.
- `packer/` - report writing.
- `checker/` - quote, claim, path, and index verification.
